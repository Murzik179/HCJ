<b>

Tag <b> - displays text within itself in bold. The closing tag is mandatory.

Attributes Usage:

  • id, class - attributes specifying unique identifiers.
  • lang - attribute specifying the language.
  • dir - attribute indicating the direction of the text.
  • title - attribute that displays the title when the cursor is hovered over.
  • style - attribute displaying information about built-in styles.
  • onclick, ondblclick, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout, onkeypress, onkeydown, onkey - attributes indicating internal events.

Example:

<p><b>Hello, World!</b></p>
